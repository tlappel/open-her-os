## How to Add Yours

1. Create your character card (see [guides/CREATING_YOUR_OWN.md](../guides/CREATING_YOUR_OWN.md))
2. Test it for at least a few conversations
3. Submit a PR with your card in a new folder under `community/characters/YOUR_NAME/`
4. Add an entry to this file with the format below
